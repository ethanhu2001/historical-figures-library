# Any Figure can interrupt a Session to ask the user a clarifying question

We considered three shapes for clarifying questions: Convener-only pre-flight (ask everything up front, then run the debate uninterrupted), Convener-only but able to interrupt mid-debate, and any Figure able to pause the debate at any point to address the user directly. We chose the last, most naturalistic option. This means a Session is not a single batch request/response with a streamed transcript — it's a suspend/resume interaction where execution can pause at any point, wait for user input, and resume from there. This is a materially larger build than pre-flight-only clarification, but it matches the truth-seeking standard set for Sessions: Figures reasoning with genuinely incomplete information should be able to ask for what they're missing when they notice they're missing it, not only before they've heard the question debated.
