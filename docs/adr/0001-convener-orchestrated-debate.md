# Convener-orchestrated debate, single model per Figure

We considered three shapes for a Session: a fixed round-robin protocol, a Convener agent that decides speaking order and when to end, and Figures self-organizing like a group chat. We chose the Convener model — a naive design would either blend Figure responses into one synthesized answer or use a rigid turn order, but the value of a "council" is Figures responding to specific points from one another, which needs a central agent making judgment calls about who speaks next. We also chose a single Claude model for every Figure and the Convener, distinguished only by system prompt, rather than a provider-agnostic abstraction — there's no evidence yet that different Figures need different underlying models, and that flexibility can be added later without disturbing this shape.
